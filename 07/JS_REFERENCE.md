## 変数の宣言

変数は使う前に宣言しなければならない。

```js
let 変数名;
```

初期値ありの場合:

```js
let 変数名 = 初期値;
```

変数名には **半角英数のみ** を用いることを推奨する。
それ以外に決まりは無い。**自分が分かりやすい名前**を付けよう。

**宣言は最初の一回だけ**必要。
**同じ変数名を複数回宣言してはいけない**。

> 例:
>
> ```js
> let xyz;
> ```
>
> ```js
> let abc = 123;
> ```



## 変数への代入

変数に入れた値は後から変更できる。

```js
let abc = 0; // 宣言
abc = 1;     // 代入
abc = 123;   // 代入
```

>  **注意:**
> **`let` を使うのは変数の宣言時のみ。**
> 宣言済みの変数を参照する時は `let` は使わない。



## 四則演算

> 足し算:
> ```js
> let a = 1 + 2; // a = 3
> ```

> 引き算:
> ```js
> let b = 3 - 4; // a = 1
> ```

> 掛け算:
> ```js
> let c = 5 * 2; // a = 10
> ```

> 割り算:
> ```js
> let d = 6 / 3; // a = 2
> ```



### 省略記法

> インクリメント
>
> ```js
> 変数名++; // 変数に +1
> ```
>
> デクリメント
>
> ```js
> 変数名--; // 変数に -1
> ```
>
> 加算代入
>
> ```js
> 変数名 += 数値; // 変数に数値を加算する
> ```
>
> 減算代入
>
> ```js
> 変数名 -= 数値; // 変数に数値を減算する
> ```



## 文字列

文字列は **`'（シングルクォート）` か `"（ダブルクォート）` で囲む**。

> 例: 変数の初期値に文字列を使う
>
> ```js
> let msg = 'こんにちは';
> ```
>
> ```js
> let msg = "こんにちは";
> ```



## 文字列の連結

**文字列に足し算**すると、**連結されて一つの文字列**になる。

```js
let greet = 'こんにちは';
let name  = '太郎';
let msg   = greet + name + 'さん'; // こんにちは太郎さん
```



## 関数

関数とは、**入力と出力を持った機械仕掛けの箱**。
**値を入力すると作動**し、**計算結果を出力**してくれる**便利な道具**。



関数の呼び出し方:

```js
関数名();
```

関数に値を入力する場合:

```js
関数名(入力値);
```

関数の出力（**返り値**）を受け取る場合:

```js
let abc = 関数名(); // 変数:abc に関数の結果が代入される
```



代表的な関数として **`alert()` 関数** が存在する。
`alert()` 関数は **入力値（引数）をポップアップ表示** してくれる関数だ。

> 例: alert 関数
>
> ```js
> alert('Hello World!'); // 「Hello World!」とポップアップ表示される
> ```



## 関数を定義する

自分専用の関数を作る:

```js
function 関数名() {
  // 処理の内容
}
```

引数有りの場合:

```js
function 関数名(引数名) {
  // 処理の内容
}
```

返り値有りの場合:

```js
function 関数名() {
  // 処理の内容
  return 返り値;
}
```



> 例: 引数を 2 乗して返す関数
>
> ```js
> function square(number) {
>   	return number * number; // 返り値
> }
> ```
>
> ```js
> let a = square(2);   // a = 4
> let b = square(3);   // b = 9
> let c = square(100); // c = 10000
> ```



## 条件分岐: if

条件によってプログラムの処理内容を変えたい場合は **`if` 文** を使う。

```js
if (条件) {
  // 実行内容
}
```

**`条件`** を満たす場合のみ `{ }` 内のコードが実行される。
`条件` には **等号**, **不等号** を用いた **比較式** を記述する。

| 比較式 | 意味 |
| ------: | ----------------- |
| `A < B` | A は B より小さい |
| `A > B` | A は B より大きい |
| `A == B` | A は B と等しい |
| `A <= B` | A は B 以下 |
| `A >= B` | A は B 以上 |
| `A != B` | A は B と等しくない |



> 例: if 文
>
> ```js
> if (age < 20) {
>   alert('お酒は二十歳になってから！');
> }
> ```



## 条件分岐: else

`if` 文の条件を**満たさなかった**場合に別の処理を実行させたい場合は **`else` 文** を用いる。

```js
if (条件) {
  // 実行内容A
} else {
  // 実行内容B
}
```

`条件` が **真 (true)** なら **`実行内容A`** が、
`条件` が **偽 (false)** なら **`実行内容B`** が実行される。



>例: if else 文
>
>```js
>if (age < 20) {
>  alert('お酒は二十歳になってから！');
>  
>} else {
>  alert('🍷乾杯！');
>}
>```



## 条件分岐: else if

`if` 文の後に **`else if`** 文を繋げると **条件分岐を追加** をすることができる。

```js
if (条件A) {
  // 実行内容A
} else if (条件B) {
  // 実行内容B
}
```

**`条件A`** が **真 (true)** なら **`実行内容A`** が実行される。

**`条件A`** が **偽 (false)** で、なおかつ
**`条件B`** が **真 (true)** なら **`実行内容B`** が実行される。



> 例: if else if 文
>
> ```js
> if (age < 20) {
>   alert('お酒は二十歳になってから！');
>   
> } else if (age == 20) {
>   alert('🍺成人おめでとう🎉');
>   
> } else {
>   alert('🍷乾杯！');
> }
> ```



## ループ: for

同じ処理を一定の回数、**繰り返し**実行させたい場合は **`for` 文** を用いる。

```js
for (let i = 0; i < 実行回数; i++) {
  // 実行内容
}
```

**変数 `i`** は **ループカウンタ**と呼ばれる。
値は `0` から始まり、`{ }` 内が実行される度に `+1` 増える。



> 例: 文字列の繰り返し
>
> ```js
> let ora = '';
> for (let i = 0; i < 12; i++) {
>   ora = ora + 'オラ';
> }
> alert(ora);
> ```



> 例: ループカウンタの利用
>
> ```js
> for (let i = 1; i < 101; i++) {
>   alert('🐑が' + i + '匹');
> }
> ```



## ループ: while

ループの回数が不定の場合は **`while` 文** の方が適している。

```js
while (条件) {
  // 実行内容
}
```

**`条件`** が **真 (true)** の場合は `{ }` の処理を**繰り返し実行し続ける**。
`条件` が **偽 (false)** になると**ループは終了する**。

また、**`break` 文** でループを**強制的に終了させる**こともできる。

```js
while (条件) {
  // 実行内容
  break; // 強制終了
}
```



> 例: カウントダウン
>
> ```js
> let i = 10;
> while (i >= 0) {
>   alert('ロケット発射まで、' + i);
>   i--;
> }
> alert('🚀');
> ```



> 例: break
>
> ```js
> while (true) {
>   let dice = ceil(random(1, 6)); // 1 から 6 までの乱数
>   alert('出目は' + dice);
>   
>   if (dice >= 4) {
>     break; // 4 以上ならループ終了
>   }
> }
> ```



## オブジェクト

オブジェクトは複数の **属性値（プロパティ）**から構成される、**データのひとかたまり**。
数値や文字列と同様、変数に代入することができる。

```js
let 変数 = {
  プロパティA: 値,
  プロパティB: 値,
  プロパティC: 値
};
```

プロパティ値を参照するには **`変数名.プロパティ名` ** という具合に記述する。



> 例: オブジェクトの定義
>
> ```js
> let student = {
>   name:   '田中太郎',
>   gender: 'male',
>   age:    16,
>   club:   '演劇部'
> };
> ```



> 例: プロパティの参照
>
> ```js
> alert('おはようございます' + student.name + 'さん');
> alert(student.club + 'へようこそ！');
> ```



> 例: プロパティ値の変更
>
> ```js
> student.age++; // インクリメント
> alert(student.age + '才の誕生日おめでとう🎉');
> ```



## 配列 (Array)

配列は**特殊なオブジェクトの一種**。
**複数の値** を **貨物列車のように連結** し、まとめて管理することのできる仕組み。

```js
let 変数名 = [値A, 値B, 値C];
```

配列に含まれている個々の値のことを **要素 (Element)** と呼ぶ。
各要素は **`0` から始まる連番** で管理されている。
要素に振られた番号のことを **添え字 (Index)** と呼ぶ。

特定の要素を参照したい時はこのように記述する:

```js
変数名[添え字]
```



### 要素の追加

既存の配列に **後から要素を追加** したい場合は **`push()`** 関数を使う。

```js
変数名.push(追加したい要素);
```

要素は **配列の最後尾** に追加される。



### 要素の個数

**配列の長さ（要素の総数）**を取得するには **`length`** プロパティを使う。

```
変数名.length
```



> 例: 配列の定義
>
> ```js
> let joestars = ['Jonathan', 'Joseph', 'Jotaro']
> ```



> 例: 要素の参照
>
> ```js
> alert(joestars[0]); // Jonathan（0番は最初の要素）
> alert(joestars[1]); // Joseph
> alert(joestars[2]); // Jotaro
> ```



> 例: 要素の追加
>
> ```js
> joestars.push('Josuke');
> alert(joestars[3]); // Josuke
> ```



> 例: for 文で配列の各要素に順次アクセスする
>
> ```js
> for (let i = 0; i < joestars.length; i++) {
>   alert(joestars[i]); // 全員の名前を読み上げる
> }
> ```
